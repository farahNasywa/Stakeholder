# Stakeholder Mapping System

Sistem Informasi Stakeholder Mapping berbasis web yang dikembangkan untuk mendukung pengelolaan dan analisis pemangku kepentingan pada Kontraktor Kontrak Kerja Sama (KKKS) di Badan Pengelola Migas Aceh (BPMA). Sistem ini membantu proses pemetaan stakeholder, analisis keterlibatan, serta penyajian informasi melalui dashboard interaktif.

## Fitur Utama

* Autentikasi dan Manajemen Pengguna
* Pengelolaan Data Stakeholder
* Stakeholder Profile Assessment
* Engagement Priority 
* Deep Analysis
* Dashboard dan Visualisasi Data
* Validasi Data oleh BPMA
